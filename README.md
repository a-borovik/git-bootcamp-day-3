# taskmanager

Учебный CLI-менеджер задач на Python. Позволяет добавлять задачи, просматривать список и помечать их как выполненные.

## Возможности

- Добавление задач
- Просмотр списка задач
- Отметка задач как выполненных
- Простой CLI-интерфейс

Минимальный API в `tasks.py`:

- `add_task(title)`
- `list_tasks()`

---

## Установка

```bash
git clone https://github.com/example/taskmanager.git
cd taskmanager
python3 -m venv venv
source venv/bin/activate
```

---

## Запуск

```bash
python3 -c "import tasks; tasks.add_task('купить хлеб'); tasks.list_tasks()"
```
